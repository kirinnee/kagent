# Service Tree (LPSM)

## Overview

All products and applications' resources in AtomiCloud are confined within the Service Tree for classification and data retrieval.

## When to Use

- Naming resources (Kubernetes, cloud resources)
- Creating labels and tags
- Setting cache namespacing
- Configuring CI/CD inputs
- Any time LPSM, LCPSM, Landscape, Platform, Service, or Module are mentioned
